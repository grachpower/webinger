# Webinger

## Test your web app
 Load testing cli tool for web apps
 
## Instalation
install globally
```
npm install -g webinger
```

## Usage
```
webinger --url=http://yourwebsite.com --method=get --rps=20 --requests=200
```

```
default:
 |- URL = required
 |- method = GET
 |- rps(requests per second) = 1
 |- requests(total requests count) = 100
```
